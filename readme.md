\# 🛍️ Customer Shopping Behavior Analysis



\## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using \*\*Python, SQL, and Power BI\*\* to identify purchasing trends, customer preferences, revenue insights, and business patterns from retail transaction data.



The project demonstrates a complete \*\*Data Analytics workflow\*\*, including:

\- Data Cleaning

\- Exploratory Data Analysis (EDA)

\- SQL Query Analysis

\- Data Visualization

\- Interactive Dashboard Creation

\- Business Insight Generation



This project was developed to strengthen my practical skills in Data Analytics and Business Intelligence using real-world retail data.



\---



\# 🎯 Business Objective

Retail companies collect large amounts of customer transaction data every day.  

The objective of this project is to analyze shopping behavior and generate insights that can help businesses improve decision-making.



This project answers important business questions such as:

\- Which product categories generate the highest revenue?

\- Which customer groups spend the most?

\- How do discounts and promo codes influence purchases?

\- Which payment methods are preferred by customers?

\- What are the seasonal shopping trends?

\- How does subscription status affect customer spending behavior?

\# 📊 Key Insights



Some important insights generated from the analysis:



✅ Clothing and Accessories categories generated high sales revenue.



✅ Customers using promo codes showed higher purchase frequency.



✅ Young adult customers contributed significantly to total purchases.



✅ Subscription customers had better repeat purchase behavior.



✅ Seasonal trends influenced product category sales.



✅ Online payment methods were more preferred by customers.
